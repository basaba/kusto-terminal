✅ WHAT YOU NEED TO KNOW
───────────────────────────────────────────────────────────────────────────────────────────────────

1. IConsoleDriver is an INTERFACE (C# interface, not abstract class)
   └─ You must provide EVERY property and method

2. Cursor management is SPLIT:
   └─ Move() sets internal position only
   └─ UpdateCursor() moves the actual terminal cursor

3. Rendering is BUFFER-BASED:
   └─ Application writes to Contents[rows, cols]
   └─ You read Contents and write to terminal in Refresh()

4. Color handling is YOUR RESPONSIBILITY:
   └─ MakeColor() converts Colors to Attributes with quantization
   └─ Handle both 16-color and TrueColor modes
   └─ Respect Force16Colors preference

5. Events must fire at the RIGHT TIMES:
   └─ SizeChanged → when terminal is resized
   └─ Refreshed → after Refresh() completes
   └─ ClearedContents → after clearing buffer
   └─ KeyDown/KeyUp → keyboard input received
   └─ MouseEvent → mouse input received
